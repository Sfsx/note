# 椭圆曲线加密算法（Elliptic Curve Cryptography）

椭圆曲线加密算法（ECC）是基于椭圆曲线数学理论实现的一种非对称加密算法

$$E:y^2 = ax^3 + bx^2 + cx + d$$
